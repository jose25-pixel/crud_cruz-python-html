# CRUD Flask App

Proyecto demo de CRUD básico con Flask.

## Estructura

- `app.py`: aplicación principal Flask.
- `routes/home.py`: rutas de la app.
- `store.py`: funciones de manipulación de datos (almacenamiento simulado en memoria o archivo).
- `templates/home.html`: plantilla principal.
- `static/css/style.css`: estilos.

## Requisitos

- Python 3.8+
- Flask

Instalar dependencias:

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python app.py
```

Abrir en el navegador: `http://127.0.0.1:5000/`

## Descripción

CRUD simple (crear, leer, actualizar, eliminar) usando Flask con rutas separadas y plantilla básica.
